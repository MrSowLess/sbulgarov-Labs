# sbulgarov Labs

Лабораторные работы по Django и Python.

## Структура проекта

- `lab2/` - Первая веб-страница с использованием Django
- `lab3/` - Блог (архив статей)
- `lab4/` - Блог с детальным просмотром статей
- `sbulgarov_lab/` - Основной проект Django

## Установка и запуск

Для каждого проекта Django:

```bash
cd lab2/firstwebpage  # или lab3/blog, lab4/blog
python manage.py runserver
```

## Требования

- Python 3.x
- Django 5.2.8
